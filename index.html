Tools
Term What it actually means
Claude Chat Claude that you have conversations with — brainstorm, write, think out loud. Think of it like texting a really smart friend who never gets tired of your questions. Browser, mobile, desktop app
Claude CoWork Claude that can actually do things — touch your files, open your browser, build real deliverables. Think of it like having an assistant who can sit down at your computer and handle the task for you. Desktop app
Claude Code Claude that helps you write, edit, and understand code and files in your project. Think of it like having a developer sitting next to you, explaining everything in plain English. Desktop app · Terminal inside VS Code
Claude Chrome Extension Claude that travels with you while you browse — highlight text, ask questions, summarize pages without switching tabs. Think of it like having Claude look over your shoulder while you browse. Chrome
Visual Studio Code (VS Code) A text editor for viewing and editing code files — has a built-in Terminal so you don't need to open a separate one. Think of it like a home base for your project files — everything in one place, easy to navigate. Desktop
Terminal Where you type commands that do things — install stuff, create files, navigate folders. Think of it like a direct line to your computer — instead of clicking through menus, you type exactly what you want it to do. Inside VS Code · Standalone app
Typora A markdown editor that makes .md files easy to read and edit. Think of it like opening a markdown file in Word instead of Notepad — clean and readable instead of full of symbols. Desktop
